
# Hospital Management System (Java + Spring Boot)

Simple hospital management system built using:

- Java
- Spring Boot
- HTML
- CSS
- JavaScript

## Features

- Add patient
- View patient list
- Web interface
- REST API backend

## Run the Project

Requirements:
- Java 17+
- Maven

Run:

mvn spring-boot:run

Open browser:

http://localhost:8080
